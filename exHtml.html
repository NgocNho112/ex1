Lệnh git branch trong Git được sử dụng để hiển thị danh sách các nhánh hiện có trong repository của bạn hoặc để tạo ra một nhánh mới. Các tùy chọn chính của lệnh git branch bao gồm: git branch: Hiển thị danh sách các nhánh hiện có trong repository. git branch : Tạo một nhánh mới với tên là . git branch -d : Xóa một nhánh đã tồn tại có tên là . git branch -m : Đổi tên một nhánh từ sang . git branch -a: Hiển thị tất cả các nhánh có trên local và remote. Lệnh git branch thường được sử dụng để quản lý các nhánh trong repository, bao gồm tạo, xóa và đổi tên các nhánh, cũng như chuyển đổi giữa các nhánh khác nhau.
